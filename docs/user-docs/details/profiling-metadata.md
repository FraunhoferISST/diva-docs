# Profiling Metadata

Currently we support automatic analysis of 5 different file types.

+ `application/pdf`
+ `text/plain`
+ `text/csv`
+ `application/x-sas-data`
+ `image/jpeg`

## Text Files

+ `application/pdf`
+ `text/plain`

## Table Files

+ `text/csv`
+ `application/x-sas-data`

## Image Files

+ `image/jpeg`
