Import and Create

This page is used to populate the data catalog with entities. Main entities are currently resources and assets.

tip

In the future you might encounter other entities like services or users themselves.

  • To create a resource, click on the Import new resources card.
  • To create an asset, click on the Create new asset card.
  • To create an service, click on the Add new service card.
